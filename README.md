# Landing Page - Barbosa & Costa Advocacia

Deploy: https://larasimmer.github.io/barbosa-costa-advocacia/

Este projeto foi desenvolvido com foco em demonstrar habilidades de criação de sites institucionais multi-páginas voltados à conversão e autoridade de marca, aplicando boas práticas de UX, acessibilidade e SEO para um escritório de advocacia.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível com foco em SEO.
- **CSS3**: Estilização moderna, layouts flexíveis (Flexbox e Grid), animações e design responsivo.
- **JavaScript (Vanilla)**: Lógica para o menu hamburger e interatividade da interface.
- **Font Awesome**: Ícones vetoriais para sinalização visual e acessibilidade.
- **Google Fonts**: Tipografia sofisticada (Playfair Display para títulos e Source Sans 3 para corpo de texto).
- **SVG**: Logotipos e ícones otimizados para alta performance e nitidez em qualquer tela.

## Principais Funcionalidades

- **Design Responsivo & Tablet-Ready**: Layout otimizado para dispositivos móveis, tablets (incluindo iPad Air) e desktops, com transição inteligente de menu a partir de 1024px.
- **Estrutura Multi-páginas**: Navegação completa entre Home, Quem somos, Áreas de atuação, Equipe e Contato.
- **Menu Hamburger**: Navegação mobile fluida e intuitiva com fechamento automático ao clicar fora ou em links.
- **Formulário de Contato Premium**: Estruturado com validação, focado em captura de leads jurídicos e integrado visualmente ao design.
- **Integração com Google Maps**: Localização dinâmica do escritório na página de contato.
- **Acessibilidade (A11y)**: Uso de atributos ARIA, labels descritivos e contraste de cores otimizado.

## Estrutura do Projeto

- **index.html**: Landing page principal com visão geral dos serviços e equipe.
- **quem-somos.html**: Página dedicada à história, valores e diferenciais do escritório.
- **areas.html**: Detalhamento técnico das áreas de atuação jurídica.
- **equipe.html**: Perfis detalhados dos sócios e filosofia de trabalho.
- **contato.html**: Hub de atendimento com formulário, informações de contato e mapa.
- **css/style.css**: Arquivo central de estilos, variáveis de marca e media queries.
- **js/script.js**: Lógica de comportamento do menu e interações.
- **images/**: Diretório com ativos visuais, fotografias e favicon.

## Como Executar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para uma experiência de desenvolvimento melhor, recomenda-se o uso de uma extensão como "Live Server" no VS Code para garantir o funcionamento correto de links relativos e recarregamento automático.
